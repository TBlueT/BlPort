    XPlane radio

    라즈베리파이를 이용하여 만듬

    X_Plane이라는 시뮬레이션을 실감 적으로 컨트롤 하고 싶어 라디오통신부 분을 회로, 프로그램을 제작을 했습니다.

    • 사용한 툴: OrCad, python
    • 사용 환경: 윈도우, 라즈비안

    Simulator

    지인의 회사에서 외주를 받아서 제어 프로그램 제작하였습니다.

    "yaw vr" 콤팩트 모션 시뮬레이터와 같은 형식의 시뮬레이터프로젝트를 외주를 받아 PC 제어 프로그램 부분을 코딩을 하여 다양한 게임데이터 추출과 3축 모터 제어를 합니다.

    • 사용툴 : Python, Qt5, MFC, C++
    • 사용환경 : 윈도우

    3D Print

    계속 출력속도 및 품질을 위해 개조를 하는 프로젝트

    3D Print의 출력 속도, 품질을 높이기 위하여 개조를 계속 진행 중입니다. 진행상태 : 제어보드교체 및 라즈베리파이 추가, 펌웨어교체(Klipper), 해드쿨링교체, 자동전원관리, LED바, 유리배드, 스탭모터드라이브 교체, BT스위치

    • 사용툴 : Python, 옥토프린터
    • 사용환경 : 윈도우, 라즈비안, Klipper

    Motor

    3D print로 만든 모터

    구리를 제외한 모든 부분은 3D 프린터로 출력을 하여 제작했으며 코일은 14줄씩 3층 3상 연결로 제작하였습니다.

    Ambient Light

    아두이노 및 python으로 만들었습니다.

    모니터 뒤에 RGB LED바 를 설치하였습니다. Python win32API를 활용하여 모니터 스크린을 읽고 opencv로 모니터 뒤에 있는 LED바 위치에 해당하는 구역을 분석 후 Serial통신으로 아두이노가 LED바를 제어합니다.

    • 사용툴 : Python, 아두이노, pywin32, opencv-python, 코드
    • 사용환경 : 윈도우
